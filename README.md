# Distgroup - Натуральные компоненты для производства косметики

Официальный дистрибьютор АО "Алуштинский эфиромасличный совхоз-завод" и ООО «Ароматы натуральных растений Абхазии».

## Категории продукции

- **Эфирные масла** (34 товара)
- **Гидролаты** (21 товар)
- **CO₂ Экстракты** (7 товаров)
- **Рафинированные масла** (16 товаров)
- **Нерафинированные масла** (33 товара)

## Редактирование

Откройте `index.html` в любом текстовом редакторе. Все данные находятся в JavaScript объекте `categories` в конце файла.
